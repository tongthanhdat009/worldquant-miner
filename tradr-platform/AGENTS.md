# tradr-platform

## Purpose
Trading game platform built with Next.js and NestJS. Contains backend, frontend, and PDAC (Prediction DCA) components.

## Subdirectories

| Directory | Purpose |
|-----------|---------|
| `backend/` | NestJS backend API (see `backend/AGENTS.md`) |
| `frontend/` | Next.js frontend (see `frontend/AGENTS.md`) |
| `pdac/` | PDAC (Prediction DCA) module |

## Tech Stack
- **Backend**: NestJS + Prisma
- **Frontend**: Next.js
- **Database**: PostgreSQL (via Prisma)
